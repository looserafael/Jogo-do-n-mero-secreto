<h4 align="center"> 
Projeto feito com a ajuda do curso da Alura de introdução a lógica de programação. Utilizei a linguagem javaScript com um HTML (já pronto) para interação com a linguagem.

A página consiste em um jogo que o usuário define a quantidade de números sorteáveis (através do prompt), então um número aleatório é gerado (oculto ao usuário) e então é possível "chutar" números através do botão com o mesmo nome. 

O site dará dicas se o número é maior ou menor até o usuário acertar. Após o acerto, o botão reiniciar jogo ficará habilitado para um nova rodada. Site disponibilizado via Github. Boa diversão! 
</h4>
